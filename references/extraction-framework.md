# 虚构角色提炼方法论

## 核心原则

虚构角色蒸馏与真实人物蒸馏的本质区别：

| 维度 | 真实人物（女娲） | 虚构角色（谛听） |
|------|----------------|---------------------|
| 研究对象 | 真实思维方式 | 角色设定/表演风格 |
| 信息来源 | 著作、访谈、决策记录 | 原著文本、影视素材 |
| 核心输出 | 思维操作系统（认知框架） | 角色表演脚本（沉浸扮演） |
| 追求目标 | 接近真实思维方式 | 表演一致性而非"真实性" |
| 诚实边界 | 必须明确标注 | 替换为"表演边界" |

## 信息来源价值分级

### 第一层：原著文本（最高权威）

定义：作者写的原始文本，文学作品的直接载体

处理原则：
- 经典文学（四大名著）：原著优先，影视作为补充
- 影视角色：原片剧本优先，改编作为参考

### 第二层：影视素材

定义：演员对角色的表演诠释，包括：
- 肢体语言
- 台词节奏
- 表情特征
- 声音特征

处理原则：
- 是原著的补充，不是替代
- 需要区分"原著设定"和"演员发挥"
- 标注表演风格来源（哪个演员/哪个版本）

### 第三层：同人创作

定义：fan对角色的二次创作和深度解读

处理原则：
- 仅作扩展视角，不作为主要依据
- 区分"有深度的分析"和"纯粹娱乐创作"
- 标注来源和可信度

### 第四层：分析文章

定义：学术研究或资深fan的系统性分析

处理原则：
- 作为参考框架提取
- 验证原始来源
- 区分学术分析和主观解读

## 六路Agent调研法

### Agent 1：原著文本分析

**调研目标**：
- 角色基本设定（身份、背景、核心关系）
- 经典场景（5-10个）
- 原著中的关键转折点

**输出格式**：
```markdown
# 原著文本分析 - [角色名]

## 基本设定

### 身份信息
- 姓名：
- 身份：
- 背景：

### 核心性格（3-5个关键词）
- 关键词1：[说明]
- 关键词2：[说明]

## 经典场景

### 场景1：[场景名]
**原文摘录**：
> [摘录]

**上下文**：[场景背景]
**为什么重要**：[说明]

### 场景2：...

## 行为模式

### 模式1：[模式名]
**触发场景**：[描述]
**原著描述**：[相关原文]

## 关键转折点

### 转折1：[事件名]
**原文证据**：[摘录]
**变化分析**：[分析]
```

## 融合与校验

### 信息冲突处理

当不同来源信息冲突时，按优先级处理：
1. 原著文本 > 影视素材 > 同人创作
2. 同一来源的不同版本，标注差异

### 角色一致性校验

完成调研后，检查：
1. 各Agent输出是否一致
2. 是否存在无法调和的矛盾
3. 标注无法确定的信息

### 质量门槛

| 检查项 | 通过标准 |
|--------|---------|
| 基本设定 | 身份、背景、核心关系齐全 |
| 特质数量 | 3-7个核心特质 |
| 行为模式 | 5-10条可识别模式 |
| 台词DNA | 至少10个标志性词汇/短语 |
| 关系网络 | 至少3个主要关系 |
| 表演边界 | 至少3条明确边界 |
